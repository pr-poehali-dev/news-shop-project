# CS2 Bunny Hop Plugin

Плагин автоматического банихопа для Counter-Strike 2 на базе CounterStrikeSharp.

## Возможности

- Автоматический банихоп при удержании пробела
- Команда `/bhop` для включения/выключения
- Включен по умолчанию при спавне
- Сохранение настроек на протяжении сессии

## Установка

1. Установите [CounterStrikeSharp](https://github.com/roflmuffin/CounterStrikeSharp) на ваш сервер
2. Скомпилируйте плагин:
   ```bash
   dotnet build -c Release
   ```
3. Скопируйте `BhopPlugin.dll` в папку `addons/counterstrikesharp/plugins/BhopPlugin/`
4. Перезапустите сервер или используйте `css_plugins reload`

## Команды

- `!bhop` или `css_bhop` - Включить/выключить автобанихоп

## Требования

- CounterStrikeSharp v1.0.272 или выше
- .NET 8.0
- Counter-Strike 2 Dedicated Server

## Настройка

По умолчанию bhop включен для всех игроков при спавне. Игроки могут отключить его командой `/bhop`.

## Техническая информация

Плагин перехватывает событие прыжка и автоматически применяет вертикальный импульс (301.993377f), что позволяет делать идеальные прыжки при удержании пробела.
